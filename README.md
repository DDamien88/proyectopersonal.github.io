# TecnoEducación

##Enlace: https://ddamien88.github.io/proyectopersonal.github.io/

## Descripción
TecnoEducación es un sitio web dedicado a ofrecer cursos y capacitación en tecnología de la información.
Nuestra institución proporciona una experiencia de aprendizaje única para adquirir habilidades en computación, 
desde introducción básica hasta certificaciones avanzadas en áreas como redes y seguridad informática.

## Tecnologías Utilizadas
- **HTML5**: Estructura del sitio web.
- **CSS3**: Estilos personalizados.
- **Bootstrap**: Framework para diseño responsivo.
- **FontAwesome**: Iconos y elementos visuales.
- **JavaScript**: Interactividad (jQuery, Popper.js incluidos en Bootstrap).
- **AOS (Animate On Scroll)**: Animaciones al hacer scroll.

## Estructura del Sitio Web

El sitio web consta de las siguientes páginas principales:

1. **index.html** - Página de bienvenida con video de fondo y enlace para ingresar.
2. **pagina_principal.html** - Página principal con navegación, carrusel de imágenes y descripción de servicios.
3. **planes.html** - Página de planes y precios de los cursos.
4. **inscripcion.html** - Página de contacto para inscripciones.
5. **quienes_somos.html** - Información sobre el equipo y la institución.
6. **windows_avanzado.html** - Detalles del curso de Windows avanzado.
7. **Proximamente.html** - Página para cursos próximamente disponibles (Arquitectura de PC, Montaje de PC, etc.).

## Descripción Paso a Paso del Sitio Web

### Paso 1: Acceso Inicial
- La página de inicio (`index.html`) muestra un video de fondo con un mensaje de bienvenida: "¡Bienvenido(a) a TecnoEducación!".
- Se motiva al usuario a ingresar haciendo clic en el enlace "INGRESAR" que lleva a la página principal.

### Paso 2: Navegación en la Página Principal
- En `pagina_principal.html`, encontrarás la barra de navegación con:
  - Logo de TecnoEducación.
  - Enlaces a: Planes, Contáctanos, y un menú desplegable de Cursos.
- Un carrusel de imágenes destacando beneficios como corrección de proyectos prácticos, tutoría personalizada y clases de prueba gratuitas.
- Descripción general de la institución y sus servicios.

### Paso 3: Explorar Planes
- Desde la navegación, haz clic en "Planes" para ir a `planes.html`.
- Aquí se muestran los diferentes planes de cursos disponibles, incluyendo precios y opciones de inscripción.

### Paso 4: Contacto e Inscripción
- Haz clic en "Contáctanos" en la navegación para acceder a `inscripcion.html`.
- Esta página contiene un formulario o información de contacto para inscribirse en los cursos.

### Paso 5: Conocer al Equipo
- Accede a "Quiénes somos" desde la navegación (nota: parece que no hay enlace directo en la barra, pero la página existe).
- En `quienes_somos.html`, se presenta al equipo con imágenes y descripciones sobre la creación de experiencias de aprendizaje.

### Paso 6: Explorar Cursos
- En el menú desplegable "Cursos", selecciona un curso disponible:
  - "Windows": Lleva a `windows_avanzado.html` con detalles del curso de Windows avanzado.
  - Otros cursos como "Arquitectura de PC", "Montaje de PC", etc., llevan a `Proximamente.html`, indicando que están en desarrollo.

### Paso 7: Funcionalidades Adicionales
- El sitio incluye un buscador en la barra de navegación para buscar contenido.
- Animaciones con AOS para una experiencia visual atractiva.
- Diseño responsivo gracias a Bootstrap, adaptable a diferentes dispositivos.
